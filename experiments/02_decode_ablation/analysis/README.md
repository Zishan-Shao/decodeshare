# H2 Decode Ablation Analysis

These scripts aggregate JSON outputs from the H2 decode-ablation and
energy-control runners.

- `summarize_disturb_cot_results.py`: LOTO/all-task result summaries.
- `summarize_disturb_cot_diagnostics.py`: output-behavior diagnostic tables.
- `summarize_energy_kmatch_outputs.py`: alpha/k-match energy-control summaries.
